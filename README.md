🌦️ Weather App

A responsive Weather Application built using HTML, CSS, and JavaScript.
This app fetches real-time weather data from a Weather API and displays temperature and weather conditions based on the city entered by the user.

🚀 Live Demo
🔗 https://weather-api-project-gamma.vercel.app/

📌 Features:-

● 🔍 Search weather by city name
● ☁️ Displays weather conditions (Clear, Rainy, Cloudy, etc.)
● ⚠️ Handles invalid city input with proper error messages
● 📱 Fully responsive design

🛠️ Tech Stack

● HTML5
● CSS3
● JavaScript (Vanilla JS)
● Weather API
